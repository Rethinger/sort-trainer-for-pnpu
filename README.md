# Визуализация сортировок

Небольшое демо для сравнения популярных алгоритмов: вставки, прямой выбор, пузырёк, Шелла и быстрая сортировка Хоара.

## Запуск

1. Установите зависимости (не требуются).  
2. Откройте `index.html` в браузере или поднимите простой сервер (`npx serve .`).
3. Введите числа через пробелы или запятые, выберите алгоритм и нажмите «Сортировать».

Скрипт работает без сборки и внешних библиотек.

## Что внутри

- Пошаговый вывод с ограничением на первые шаги для больших массивов.  
- Метрики: время выполнения, количество перестановок и сравнений.  
- Генератор случайного массива и чистка формы.  
- Тёмная тема с акцентами и картами результатов.
