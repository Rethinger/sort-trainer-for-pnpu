# Візуалізація сортувань

Невелике демо для порівняння популярних алгоритмів: вставки, прямий вибір, бульбашка, Шелл і швидке сортування Гоара.

## Запуск

1. Встановіть залежності (не потрібні).  
2. Відкрийте `index.html` у браузері або підніміть простий сервер (`npx serve .`).
3. Введіть числа через пробіли чи коми, оберіть алгоритм і натисніть «Сортувати».

Скрипт працює без збірки та зовнішніх бібліотек.

## Що всередині

- Покроковий вивід з обмеженням перших кроків для великих масивів.  
- Метрики: час виконання, кількість перестановок і порівнянь.  
- Генератор випадкового масиву та очищення форми.  
- Темна тема з акцентами й картками результатів.
